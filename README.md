# Buddy-Alocator
Implementacao do Buddy Alocator utilizando  arvores 

para rodar (INTELIJ)

use um arquivo txt para benchmark e va em edit configurations, e coloque o nome do arquivo com src/ antes na aba Program Arguments:

para formatar o arquivo de entrada basta seguir o arquivo de exemplo ja disponibilizado aqui no github (programas.txt)

utilizei arvores binarias como a estrutura principal
os blocos sao modelados de maniera simples ( livre, ocupado, dividido)
potencia de 2 foi utilizada para os blocos
escolhi a alocacao first fit dando preferencia pro filho esquerdo
